If you got here from my Medium post, these are the exact instructions. Good luck!

  1. Download the provided zip file from github. It contains the pdf application and the images required by UTC.
  2. Unzip the file and edit the PDF to replace the names. Rezip the file.
  3. Attach the zip to an email and send it to "docsubmit@unicode.org" with the message body: "Please consider the attached documents for submission to the UTC."
